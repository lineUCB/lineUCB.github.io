Feature Matching

To match feature descriptors, we calculate the nearest neighbors (1-NN and 2-NN) for each feature patch. Think of the feature patches as 64-dimensional vectors, and we use Euclidean distance to find the closest matches via a nearest-neighbor search library. Then, we divide the distance to the 1-NN by the distance to the 2-NN. If the ratio is below a certain threshold, we consider the 1-NN to be a valid match. Even after preprocessing, some features may still be matched incorrectly. To address this, we can use RANSAC to remove "outlier" correspondences that don't match well. To show how this works, we can plot the feature correspondences between two images before and after applying RANSAC.

Image Stitching

Image stitching is the process of combining multiple images with overlapping fields of view to produce a single panoramic image. The stitching process can be broken down into three main steps: feature detection, feature matching, and image blending.
