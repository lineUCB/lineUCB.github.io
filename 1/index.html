Welcome to Project 1

Overview

In this project, using 3 versions of the same image, each with a color filter: red, blue and green. These when extracting the color channels in the black and white image, can be overlaid to create a color rendition of the original picture. Using different alignment algorithms such as L2 Norm and Normalized Cross-Correlation allows for a pixel alignment to properly place the images and overlay them. This is based on the concept developed by Sergei Mikhailovich Prokudin-Gorskii. He used 3 exposures of a scene which were put onto glass with either a blue, red, or green filter. When the glass plate negatives are layered they provide a colored image.

Method

To begin the program devides the image into three equal parts. Next, the parts need to be aligned. For this project I implemented Euclidian Norm(L2 Norm) and Normalized Cross-Correlation (NCC) for the Naive implementations. For this the alignment, it works through exhasitve search over a window of [-15,15] pixels. Then it calculates the best displacement between the images before returning the displacement. After that the red and green images are both adjusted to be properly aligned with the blue before they are all overlaid to return the final image. As the naive implementation is less efficient, using an image pyramid will allow the process to be sped up. Image pyramid reduces the scale of the image until a certain threshold. Next, it runs the aligment on the coarser image, before going up through the higher resolution versions of the images and just scaling the displacement to make the calculation quicker. To speed up the process.

Obstacles

I faced an issue with the Image Pyramid method on the Train image. After attempting to fix the image pyramid to work well for this image, I decided to take another approach using Edge Detection and Equalization which proved to solve the problem with the train image. After attempting with different methods of alignment, such as phase cross correlation. However, the alignment still wasn't looking better. I then implemented the image pyramid with phase cross corrolation as well as Canny Edge detection and Equalizing. This ended up fixing the issues with the train image, and also created better resolution and alignments for a lot of the other images.

Bells and Whistles

For this I implemented edge detection as well as a contrasting function. The edge detection uses equalized images to create a better alignment by using a filter that reduces the noise in the image which allows for a better alignment. The contrasting function uses the percentile to create a threshold for the pixel contrast. This was an attempt to help create better contrast in the images.
